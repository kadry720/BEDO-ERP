# BEDO Platform

`bedo_platform` is the Frappe v15 foundation app for BEDO internal enterprise workflows. This phase intentionally implements only platform structure, LDAP-only authentication, user administration, role catalog metadata, dashboard routing, empty department dashboards, audit logging, and tests.

Business workflow DocTypes are intentionally absent in this phase. ARD dashboard pages are placeholders only; ARD process logic must be derived from Chapter 4 and Chapter 6 text in future phases, not from the incorrect visual flowcharts.
